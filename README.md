\# MiniMarketManager



\## Descripción

MiniMarketManager es un sistema en consola desarrollado en Java que permite registrar productos, realizar ventas, buscar productos y calcular descuentos e IVA dentro de un mini mercado.



\## Lenguaje

Java — desarrollado en Apache NetBeans IDE



\## Funciones



\- Registrar productos: permite guardar productos con nombre, precio y stock.

\- Mostrar productos: muestra todos los productos registrados.

\- Buscar productos: permite buscar productos por nombre.

\- Realizar ventas: descuenta productos del stock y acumula ventas.

\- Calcular descuento: calcula el precio final aplicando un porcentaje de descuento.

\- Calcular IVA: calcula el precio final agregando IVA al producto.



\## Como ejecutar



1\. Abrir el proyecto en Apache NetBeans.

2\. Ejecutar la clase principal:

MiniMarketManager.java
